# Project Demo & Usage Examples

This document demonstrates how the Gemini-powered AI agents behave
with real user inputs and outputs.

---

## Demo 1: Content Roast AI

### User Input
- Screenshot of a social media profile or post

### AI Behavior
- Analyzes visual content
- Generates a light, sarcastic roast
- Switches tone to constructive feedback
- Provides actionable suggestions

### Sample Output
**Roast:**  
"This bio has the energy of 'I’ll update this later' — except later never came."

**Advice:**  
- Add a clearer value proposition  
- Improve profile picture clarity  
- Use consistent visual branding

## Demo 2: Tech Navigator AI

### User Input
- Beginner student interested in AI and frontend development

### AI Behavior
- Asks structured questions
- Classifies experience level
- Generates a personalized roadmap
- Suggests next steps and resources

### Sample Output
**Roadmap Overview:**
1. Learn programming fundamentals
2. Explore frontend basics (HTML, CSS, JS)
3. Introduction to AI concepts
4. Build small projects
5. Prepare for internships or hackathons
